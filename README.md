# Multi-omics Spatiotemporal Atlas Defines Malignant Subtype Architecture and FOSL1-Gated PLAU-PLAUR Axis in Glioblastoma Evolution

This repository contains the analysis scripts and workflows for the study titled: **"Multi-omics Spatiotemporal Atlas Defines Malignant Subtype Architecture and FOSL1-Gated PLAU-PLAUR Axis in Glioblastoma Evolution"**. 

In this study, we integrated single-cell RNA-seq, single-cell ATAC-seq, and Spatial Transcriptomics to dissect the spatiotemporal evolution of GBM malignant subtypes and identified the PLAU-PLAUR axis as a key therapeutic target to disrupt the tumor-microenvironment symbiosis.

- **In Silico Perturbation**: Virtual knockout (scTenifoldKnk) predicting the collapse of the mesenchymal network upon PLAUR silencing.
