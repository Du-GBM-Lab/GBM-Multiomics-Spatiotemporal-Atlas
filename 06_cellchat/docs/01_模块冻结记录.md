`07_细胞通讯` 已完成主分析、纯 TME sensitivity、PLAU/PLAUR 表达验证、正文/补充候选图和 source data 输出。模块进入冻结状态：不重跑 CellChat，不覆盖结果对象，不继续在本模块内扩展新的机制分析。

## 当前结论

1. 主分析采用 `CellChat输入_四亚型加纯TME.qs2`，保留四个 malignant subtype 与纯净 TME 群，排除 glial/neural-like 与 ambiguous 细胞。
2. `PLAU` pathway 在 full TME 与 pure TME 两版中均排第 17。它不是全局最强通讯通路，应写作 focused hypothesis / nominated axis，而不是 dominant TME signal。
3. `PLAU` pathway 的 LR 构成只有 `PLAU_PLAUR`。
4. pure TME 版中最强方向集中在 myeloid 与 MES-V：
   - Macrophages -> Macrophages
   - Microglial -> Macrophages
   - Malignant_MES-V -> Macrophages
   - Macrophages -> Microglial
   - Microglial -> Microglial
   - Malignant_MES-V -> Microglial
5. 表达验证显示：`PLAU` 在 Malignant_MES-V、Microglial、Macrophages 均较高；`PLAUR` 明显偏向 myeloid，恶性细胞中以 Malignant_MES-V 最高。
6. 因 `PLAUR` 受体端强烈偏向 myeloid，正式措辞固定为：`asymmetric, myeloid-converging PLAU-PLAUR axis`。不要写 `bidirectional loop`。

## 边界

CellChat 是 ligand-receptor 共表达推断，只能支持 `nominated / inferred`。本模块不证明 PLAU-PLAUR 的因果功能，不写髓系招募因果，不把 FOSL1-PLAUR 机制放在 CellChat 里证明。

后续叙事链为：CellChat 提名 MES-V/myeloid PLAU-PLAUR axis -> SCENIC 只读支持 FOSL1 是 MES-V-associated candidate -> ATAC 检查 PLAUR regulatory accessibility / AP-1 motif -> 湿实验验证肿瘤侧 FOSL1 -> PLAUR。
