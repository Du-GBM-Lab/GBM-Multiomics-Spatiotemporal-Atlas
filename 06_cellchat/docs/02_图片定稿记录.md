`<DATA_ROOT>/项目/分型/修稿杠生信/图片表格/06_细胞通讯/source_data/四亚型配受体差异散点图_source.csv`

统计口径：

- 对 LR dotplot 中同一批 ligand-receptor pairs，按 malignant subtype 汇总其指向全部 TME 的 total communication probability。
- 颜色：该 LR 在该亚型中的 total communication probability。
- 点大小：该 LR 在该亚型中相对另外三个亚型平均值的 enrichment。

差异指标：

`log2((本亚型 total probability + 1e-6) / (其他三个亚型平均 total probability + 1e-6))`

解释：

- 该指标回答“这个 LR 是否偏向某个亚型”，不是回答该 LR 的绝对强度。
- 点大小只使用正 enrichment，并截断上限为 6，避免其他亚型为 0 时图例被极端值撑开。
- `PLAU -> PLAUR` 在 `MES-V` 中 `total_prob = 0.000995`，其他三个亚型为 0；因此 MES-V enrichment 很高，截断后点大小为 6。
- 当前排版尺寸为 4.05 x 3.25 inch，比最紧凑版本稍微放松，避免点和文字过挤。
- LR 行按四个亚型的 `log2_enrichment_vs_others` 模式做层次聚类，列顺序固定为 `NPC-P / OPC-M / MES-V / MES-I`。
