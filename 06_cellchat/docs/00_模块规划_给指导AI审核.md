## 2. 原稿思路复盘

原稿把机制链写成：

`FOSL1 / TEAD4 -> PLAUR / MMP9 -> EMT / invasion -> TAM recruitment / immunosuppressive niche`

这条链的问题是：旧分析从 TF/SCENIC 网络里强行突出 `FOSL1-PLAUR`，再往微环境解释，容易被审稿人质疑为“先有机制图，再找证据”。旧 SCENIC 补充 notebook 里还出现过强制加入 PLAUR 边的逻辑，这种展示不适合作为正文证据。

本轮改法是把叙事入口前移到 cell-cell communication：

`MES-V / MES-I 与 TME 的 ligand-receptor 互作 -> PLAU-PLAUR / PLAUR 相关通讯是否富集 -> 再回到 TF regulon 看 FOS/AP-1/ETS/CEBP 等 regulator`

这样 PLAUR 先作为微环境互作信号被数据推出来，再讨论可能的上游 TF，逻辑更稳。

## 5. 主分析问题

第一层：全局通讯

- 哪些 malignant subtype 与 TME 的 overall interaction strength / number 更高？
- MES-V / MES-I 是否更偏向 myeloid、vascular 或 stromal-like compartments？
- incoming / outgoing pattern 是否支持 MES subtype 的 microenvironment remodeling？

第二层：PLAU-PLAUR / PLAUR focused

- CellChatDB.human 里是否存在 `PLAU-PLAUR` 或含 `PLAUR` 的 ligand-receptor interaction？
- 这些 interaction 的 source / target 是谁？
- 是否是 MES-V 或 MES-I 与 myeloid / endothelial / mural / macrophage-microglia 相关？
- PLAUR 是 malignant side receptor，还是 TME side receptor？这会决定 narrative。

第三层：和 TF 模块衔接

- 如果 PLAUR/PLAU 通讯集中在 MES-V，再回到 SCENIC：FOSL1 只能写作 MES-V-associated candidate regulator。
- 如果 PLAUR/PLAU 通讯集中在 myeloid/TME 侧，则不能把 FOSL1-PLAUR 写成 malignant-intrinsic 主轴，需要改写成 microenvironment-linked PLAUR program。
- 不管哪种结果，SCENIC 只能支持 regulon activity，不证明因果。

## 8. 科学边界

CellChat 推断的是 ligand-receptor co-expression based communication probability，不是细胞间真实配体蛋白流动，也不是因果验证。

可以写：

- `inferred intercellular communication`
- `PLAU-PLAUR signaling was nominated`
- `consistent with a PLAUR-linked microenvironmental program`

不能写：

- `PLAUR recruits TAMs`（除非后续实验或空间证据直接支持）
- `FOSL1 drives PLAUR through CellChat`
- `CellChat demonstrates causality`
