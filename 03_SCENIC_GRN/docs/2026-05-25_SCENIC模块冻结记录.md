# SCENIC 模块冻结记录

冻结日期：2026-05-25

## 冻结状态

`02_TF_regulon_SCENIC` 已完成当前轮 SCENIC 重跑，进入冻结状态。后续先开展独立的细胞通讯模块；在细胞通讯确认 PLAUR 相关叙事起点之前，本模块不再新增分析、不重跑、不覆盖既有结果。

## 当前结论

本模块支持“四亚型对应 distinct regulon activity programs”：

- `NPC-P`：proliferative / neural progenitor regulon program，代表 TF 包括 `TFDP1`, `POU3F2`, `NR2F1`, `E2F7`, `SOX2`, `E2F1`。
- `OPC-M`：OPC / myelination regulon program，代表 TF 包括 `SOX10`, `SOX8`, `SREBF1`, `POU3F3`。
- `MES-V`：vascular / stress / ETS / AP-1-like regulon program，代表 TF 包括 `NFATC4`, `CREB3L2`, `HIC1`, `ETS1`, `NR2F2`, `BHLHE40`, `ELK3`, `ERG`, `FOSL2`。
- `MES-I`：inflammatory / NF-kB / IRF / CEBP-like regulon program，代表 TF 包括 `IKZF1`, `IRF8`, `MAFB`, `SPI1`, `CEBPA`, `REL`, `RUNX3`, `NFKB2`。

`FOSL1` 目前定性为 MES-V-associated candidate，不是 master regulator。`FOSL1` 在 MES-V 的 RSS 排名为第 18，`FOSL2` 排名更靠前。本模块不再围绕 `FOSL1-PLAUR` 单独推进机制图。

- `outputs/scenic_AUCell_regulon_activity.h5ad`：28,213 cells x 115 regulons 的 AUCell matrix。
- `outputs/scenic_grn_edgelist.csv`：GRNBoost2 推断的 TF-target edgelist。
- `outputs/scenic_rss_scores_regulon_by_subtype.csv`：四亚型 RSS matrix。
- `outputs/scenic_rss_all_regulons_ranked_by_subtype.csv`：每个亚型的全 regulon RSS 排名。
- `figures/source_data/scenic_top30_regulons_by_subtype.csv`：每个亚型 top 30 regulons。
- `logs/02_run_SCENIC_stdout.log` / `logs/02_run_SCENIC_stderr.log`：SCENIC 主流程日志。
- `logs/03_compute_RSS_from_AUCell_summary.txt`：RSS 续算日志。

## 冻结原因

论文叙事将调整为“细胞通讯 -> TF regulon -> ATAC -> 实验验证”的链条。`FOSL1-PLAUR` 不再从 SCENIC 网络图强行起步，而是等待细胞通讯模块先判断 PLAUR/PLAU 相关互作是否构成合理入口。因此，本模块只保留当前数据驱动 regulon 结果，不继续向 FOSL1 下游扩展。

## 后续解冻条件

仅当细胞通讯模块明确支持 PLAUR/PLAU 相关互作，并需要与 TF regulon 结果衔接时，再回到本模块提取相应候选 TF 或绘制补充图。届时仍需使用 `putative`, `inferred`, `candidate`, `consistent with` 等措辞，避免将 SCENIC 结果写成因果证据。
